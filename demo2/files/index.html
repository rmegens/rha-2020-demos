This is a demo page for demo 2

This webpage is created by:

  • creating a Dockerfile
    • get a runtime with rhel8
    • install apache2
    • copy index.html to /var/www/html
    • start the application
  • create an image
    • podman build -t rha-may-webapp .
  • run a container from the image
    • podman run -d --name rha-mayweb rha-may-webapp

Next steps

  • show index page with Lynx
    • lynx http://localhost
  • show running processes on host
    • ps aux | grep apache
  • show running processes in container
    • podman exec -ti rha-mayweb /bin/bash
    • ps aux | grep apache
