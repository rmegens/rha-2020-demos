This is a demo page for demo 3

This webpage is created by:

  • login to Openshift Cluster
    • oc login -u developer -p [my-secret-password-here] https://myclustername:6443
    • oc new-project rha-may-web
  • create an image and run the pod
    • oc new-app httpd~http://github.com/rmegens/rha-2020-demos --context-dir=demo3/files/ --name my-webapps2i

Next steps

  • Show running pods
    • oc get pods
  • Create a route for the webapp
    • oc expose svc mywebapps2i
  • show index page with a webbrowser
    • Use your webbrowser : http://my-webapps2i-rha-demo3-test1.apps-crc.testing
  • show running processes on host
    • ps aux | grep httpd
  • show running processes in container
    • oc rsh rha-my-webapps2i
    • ps aux | grep httpd
